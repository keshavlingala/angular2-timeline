#angular2-timeline

##Install

```
> npm i angular2-timeline
```

Add Timeline Module
```typescript
import {TimelineModule} from "angular2-timeline"; 

@NgModule({
    ...
  imports: [
    TimelineModule,
    ...
  ],
```
##How to use
```html
<timeline>
  <timeline-item>
 <!-- your content here -->
  </timeline-item>
```
